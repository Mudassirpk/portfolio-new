Overview:
A market place for UAE based Logistics company where user can sell and purchase cargo assets.

## Key Features

- **Role Based Auth system**: Implemented role based authentication system for user's access.
- **Payment Gateway**: Integrated stripe payment Gateway for payments and Stripe connect for Intra-Platform payments.
- **Order tracking**: Implemented order tracking system.
- **Multiple Consoles**: Developed Multiple consoles based on user roles.

## Technologies Used

- **Reactjs**.
- **NestJS**.
- **PostgreSQL**.
- **Typescript**.
- **ShadCN UI**.
