## Overview

Cakecrush is an ecommerce experience for a UAE-based cake brand, pairing a branded storefront with a streamlined order flow.

## Key Features

- **Storefront**: Product catalog with add-to-cart and order review tuned for cakes.
- **Checkout**: Orders handled through an ExpressJS backend, capturing delivery details.
- **Responsive UI**: ShadCN UI + Material UI for cohesive theming across devices.

## Technologies Used

- **Next.js**.
- **ExpressJS**.
- **ShadCN UI**.
- **Material UI**.
- **Typescript**.
