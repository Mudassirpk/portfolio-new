## Overview

Uplift Center is a healthcare web experience for a UK clinic, highlighting services, practitioners, and booking touchpoints with a fast static delivery.

## Key Features

- **Services highlight**: Clear presentation of core treatments and programs.
- **Lead capture**: Contact and booking prompts surfaced across the site.
- **Performance-first**: Astro static output paired with Tailwind CSS for rapid load times.

## Technologies Used

- **Astro**.
- **Tailwind CSS**.
- **Typescript**.
